scala判断一个数据或者对象的类型只需要在该对象后面添加 `.getClass.getSimpleName` ：

~~~ruby
scala> 222.getClass.getSimpleName
res1: String = int

scala> "222".getClass.getSimpleName
res2: String = String
~~~
